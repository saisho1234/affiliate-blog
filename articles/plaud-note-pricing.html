PLAUD NOTE 料金体系の全体像

PLAUD NOTEの料金体系は「本体（買い切り）＋サブスクリプション」の2段階で構成されています。

料金体系のポイント

  • 本体は一度購入すれば追加費用なし（買い切り）
  • AI文字起こし・議事録生成にはサブスクが必要
  • スタータープランは月300分まで無料
  • プロプランは月払い（1,980円）または年払い（16,800円）
  • 全モデル（Note / Note Pro / NotePin / NotePin S）でサブスクプランは共通

「録音だけしたい」という場合は本体のみで使用可能ですが、PLAUD NOTEの最大の魅力であるAI議事録・文字起こしを活用するにはサブスクが必要です。まずはスタータープラン（無料）で試して、使用量が月300分を超えるようになったら有料プランへ移行するのが賢明です。

詳しい機能・レビューはPLAUD NOTE 評判・レビュー記事もご覧ください。

スタータープラン（無料）の詳細

本体購入後、追加料金なしで利用できるプランです。月300分（5時間）のAI文字起こし・要約が含まれます。週1〜2回・1時間以内の会議録音であれば月4回×60分＝240分で収まるため、スタータープランで十分対応できます。

プロプランの詳細

月1,200分（20時間）のAI文字起こし・要約が利用可能。月払い1,980円・年払い16,800円（月換算約1,400円）です。年払いにすると年間約7,560円お得になります。毎日1時間の会議を週5日録音しても月20時間（1,200分）の枠内に収まります。

無制限プランの詳細

年額40,000円（月換算約3,333円）で文字起こし・要約が無制限に利用できます。医師・弁護士・ジャーナリスト・研究者など毎日数時間の録音をするヘビーユーザー向けです。

損益分岐点シミュレーション

投資回収の計算（PLAUD NOTE＋プロプラン 初年度44,300円の場合）

  • 時給2,000円の方 → 議事録作成を22時間節約すれば元が取れる
  • 時給3,000円の方 → 議事録作成を14.7時間節約すれば元が取れる
  • 1回の会議で30分の議事録作成を削減・月4回 = 月2時間節約 → 年間24時間節約
  • 時給2,000円なら初年度でほぼ元が取れる計算（24時間×2,000円=48,000円）

どのプランを選ぶべきか

プラン選択ガイド

  • 月に会議1〜3回・1回あたり1時間以内 → スタータープラン（無料）で十分
  • 週2〜3回の会議・月10時間程度の録音 → プロプラン（年16,800円）がおすすめ
  • 毎日使う・月20時間超の録音 → 無制限プラン（年40,000円）が安心
  • チームで複数デバイス導入 → Teamプランを問い合わせ

まずはスタータープランで1ヶ月試してみて、月300分（5時間）を超えそうであればプロプランに移行する流れが最もリスクが少ない選び方です。プロプランは月払い（1,980円）でも試せるので、継続が確定してから年払い（16,800円）に切り替えると節約になります。

買う価値はあるのか？コスパ分析

「3万円近くするのに買う価値があるのか？」という疑問に正直に答えます。

コスパが高い人

  • 週2回以上の会議・商談がある営業職・コンサル・マネージャー
  • 毎回30分以上の議事録作成に時間を取られている人
  • 取材・インタビューを月複数回行うライター・記者・研究者
  • 時給換算が高い職種の人（弁護士・士業・医師・フリーランスなど）

コスパが低い人

  • 月に会議が1〜2回しかない人（スタータープランのみで十分・本体代の回収が長い）
  • すでにGoogle DocsのAI機能など無料代替手段で対応できている人
  • 初期費用3万円の出費が厳しい人（まずNottaなどアプリ型ツールを試すのもあり）

アプリ型で低コストから始めたい場合はAIメモ・ノートツール比較も参考にしてください。また無料で使えるAIツールまとめも合わせてご覧いただけます。

よくある質問

PLAUD NOTEはいくらで買えますか？
スタンダードモデルのPLAUD NOTEは約27,500円〜、上位モデルのNote Proは約30,800円が目安です。PLAUD公式サイト・Amazon・楽天市場などで購入できます。公式サイトのLINE会員登録やクーポン利用でさらに安くなる場合があります。
PLAUD NOTEのサブスクは必須ですか？
録音自体は本体のみで可能です。ただしAI文字起こし・要約・議事録生成にはサブスクリプションが必要です。スタータープランは月300分まで無料で利用できます。
PLAUD NOTEは年払いと月払い、どちらがお得ですか？
プロプランの場合、年払い（16,800円）は月払い換算（1,980円×12ヶ月＝23,760円）より年間約7,560円お得です。継続利用が確定したら年払いを選びましょう。
PLAUD NOTEをできるだけ安く買う方法はありますか？
公式サイトのLINE会員登録・クーポン適用や、Amazonのタイムセール・楽天スーパーセール時に安く購入できる場合があります。また公式サイト限定のセット割引も定期的に実施されています。
PLAUD NOTEのスタータープランでできることは何ですか？
スタータープランは月300分（5時間）まで無料でAI文字起こし・要約・議事録生成が利用できます。月に1〜2回程度の短時間会議録音であれば、スタータープランのみで十分まかなえます。
PLAUD NOTE本体の価格はモデルによって違いますか？
はい、モデルによって異なります。スタンダードの「PLAUD NOTE」が最も安く（約27,500円〜）、5マイク搭載の「Note Pro」が約30,800円、ウェアラブル型の「NotePin」「NotePin S」は公式サイトにてご確認ください。
PLAUD NOTEのサブスクを途中解約できますか？
月払いプランはいつでも解約できます。年間契約の場合の返金ポリシーはPLAUD公式の利用規約をご確認ください。サブスクをやめた後もスタータープランとして月300分は引き続き利用できます。
PLAUD NOTEは法人購入・領収書発行に対応していますか？
公式サイトでの購入時に領収書発行が可能です。法人購入・まとめ買いについてはTeamプランの問い合わせ窓口にご相談ください。
PLAUD NOTE、NotePinで料金プランは同じですか？
はい。PLAUD NOTE・Note Pro・NotePin・NotePin Sのすべてのモデルで、サブスクリプションプランは共通です。本体を変えてもサブスクはそのまま引き継げます。
PLAUD NOTEのトータルコストはどのくらいですか？
スタンダードモデル＋プロプランで計算すると、1年目は本体27,500円＋サブスク16,800円＝合計約44,300円。2年目以降はサブスクのみ16,800円/年です。月換算すると初年度は約3,692円/月、2年目以降は約1,400円/月となります。
